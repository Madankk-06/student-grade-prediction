# 🎓 Student Grade Prediction System

A Machine Learning project that predicts student final grades based on academic and personal factors.

---

## 🚀 Features

* Predict final grade (G3)
* Interactive Streamlit UI
* Performance classification (Excellent / Good / Average / Needs Improvement)
* Suggestions based on prediction

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit

---

## 📊 Machine Learning Model

* Algorithm: Random Forest Regressor
* Dataset: UCI Student Performance Dataset
* Target Variable: G3 (Final Grade)

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python train_model.py
streamlit run app.py
```
